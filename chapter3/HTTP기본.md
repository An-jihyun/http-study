# HTTP 기본

## 모든 것이 HTTP
HTTP(HyperText Transfer Protocol)

HTTP 메시지에 모든 것을 전송
- HTML, TEXT
- IMAGE, 음성, 영상, 파일
- JSON, XML (API)
- 거의 모든 형태의 데이터 전송 가능
- 서버간에 데이터를 주고 받을 때고 대부분 HTTP 사용
